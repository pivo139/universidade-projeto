## Descrição/Objetivo

O presente projeto propõe o desenvolvimento de um sistema de gestão escolar para centros universitários, através de uma plataforma integrada e intuitiva para o gerenciamento eficiente de informações acadêmicas e administrativas.

#### Atribuir Notas

Ao acessar a opção, o professor irá selecionar a matéria e a turma, ambas informações serão apresentadas na tela e com base no escolhido será apresentado uma listagem dos alunos da turma. Cada linha corresponde a um aluno e para cada um, existem três colunas para preenchimento das notas de acordo com o critério adotado pela universidade.
